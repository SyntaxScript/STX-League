# Этап 4 — Назначить себя администратором

> **Зачем**: только пользователи из таблицы `admins` могут одобрять/отклонять заявки. Это серверная проверка через RLS — обойти из браузера невозможно.

---

## Часть А. Узнать свой Discord ID

### 1. Включить режим разработчика в Discord
1. Открой Discord (приложение или https://discord.com)
2. **Настройки** (шестерёнка снизу слева у твоего ника)
3. Слева в меню → **Расширенные** (внизу списка)
4. Включи переключатель **Режим разработчика** (Developer Mode)
5. Закрой настройки

### 2. Скопировать свой ID
1. Найди себя в любом сервере или в списке друзей
2. ПКМ по своему имени/аватарке
3. В появившемся меню — внизу пункт **Копировать ID пользователя**
4. Это число вида `123456789012345678` (16-19 цифр)

---

## Часть Б. Добавить себя в admins

### Вариант 1 (рекомендую): через SQL Editor
1. Supabase Dashboard → **SQL Editor** → **+ New query**
2. Вставь (замени `ТВОЙ_DISCORD_ID` на свой ID, который скопировал):
   ```sql
   insert into public.admins (discord_id, discord_username, note)
   values ('ТВОЙ_DISCORD_ID', 'твой_ник', 'Главный администратор');
   ```
3. **Run**
4. Должно появиться `Success. 1 row inserted`

### Вариант 2: через Table Editor
1. Supabase Dashboard → **Table Editor** → таблица **admins**
2. **+ Insert** → **Insert row**
3. Заполни:
   - `discord_id` → твой ID
   - `discord_username` → твой ник (опционально)
   - `note` → `Главный администратор` (опционально)
4. **Save**

---

## Часть В. Проверить, что всё работает

1. SQL Editor → **+ New query**:
   ```sql
   select * from public.admins;
   ```
2. **Run** → должен вернуться 1 ряд (ты сам)

---

## Если нужно добавить ещё админов потом

Просто повтори Часть Б с другим Discord ID:
```sql
insert into public.admins (discord_id, discord_username, note)
values ('1234567890123456789', 'co_admin_nick', 'Помощник');
```

---

## Удалить админа
```sql
delete from public.admins where discord_id = '1234567890123456789';
```

---

## ✅ Готово, когда:
- ✅ Твой Discord ID скопирован
- ✅ В таблице `admins` есть ты
- ✅ `select * from admins` показывает тебя
