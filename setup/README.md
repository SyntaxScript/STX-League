## 🔑 Что от тебя нужно получить (порядок не важен)

После того как пройдёшь Этапы 2 и 3 — пришли мне:
1. **Supabase Project URL** (вида `https://abcdef.supabase.co`) — публичный, безопасно
2. **Supabase anon public key** (длинный JWT) — публичный, безопасно

**НЕ присылай мне:**
- ❌ `service_role` key (полный доступ к БД)
- ❌ Discord Client Secret (его вставляешь сам прямо в Supabase)
- ❌ Database password (только тебе нужен на случай восстановления)
- ❌ Discord credentials (BOT token и т.п.)
